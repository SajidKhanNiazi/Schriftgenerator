# agents.md

## Project Name
Schriftgenerator – German Font Converter (Single Page SaaS-ready Website)

## Project Goal
Build a mobile-first, SEO-optimized, single-page German "Schriftgenerator" website.
The website converts normal text into multiple Unicode font styles.
Target users are mobile users in Germany.
The project should be scalable into a SaaS product.

## Language & Market
- Primary language: German
- Target country: Germany
- All UI text, headings, and content must be in German

## Website Type
- Single Page Application (SPA-style layout)
- One main homepage with scroll-based sections
- Separate legal pages required (Impressum, Datenschutz, AGB)

## Main Keyword (SEO)
- Schriftgenerator

Secondary keywords to support:
- Instagram Schriftgenerator
- WhatsApp Schriftgenerator
- TikTok Schriftgenerator
- Schrift umwandeln
- Text in Schrift umwandeln

## Page Structure (Single Page)

### Section 1: Hero / Main Tool
- H1: Schriftgenerator – Texte online umwandeln
- Short German description (2–3 lines)
- Large text input field
- Live font output generation
- Copy-to-clipboard button
- Optimized for mobile usage

### Section 2: Font Converter Sections (Scroll Sections)
Each font category must be its own visible section with an H2 heading.

Required sections:
- Instagram Schriftgenerator
- WhatsApp Schriftgenerator
- TikTok Schriftgenerator
- Fette Schrift
- Kursive Schrift
- Fancy Schrift

Each section must include:
- H2 heading with keyword
- Short explanation (2–3 German sentences)
- List of generated font styles
- One-tap copy button for mobile users

Anchor IDs must be used:
- #instagram-schriftgenerator
- #whatsapp-schriftgenerator
- #tiktok-schriftgenerator

### Section 3: Popular Use Cases
Explain how users use the Schriftgenerator for:
- Instagram Bio
- WhatsApp Status
- TikTok Namen
This section is for SEO content and user clarity.

### Section 4: How It Works
Simple steps in German:
1. Text eingeben
2. Schrift auswählen
3. Kopieren & einfügen

### Section 5: FAQ Section
Include SEO-friendly FAQs such as:
- Was ist ein Schriftgenerator?
- Ist der Schriftgenerator kostenlos?
- Funktioniert die Schrift auf WhatsApp und Instagram?
- Muss ich etwas installieren?

## Navigation
- Sticky top navigation
- Navigation links must scroll to sections using anchor links
- Mobile hamburger menu required

## Mobile-First Requirements
- Designed for thumb usage
- Large buttons
- Fast loading
- No heavy animations
- No intrusive popups
- Sticky copy button preferred

## SEO Requirements
- Only one H1 on the page
- Proper H2 and H3 usage
- German meta title and meta description
- Semantic HTML
- Internal anchor links
- Content visible to search engines (no hidden text)

## Performance
- Optimized for fast load on mobile
- Lazy-load font generation if needed
- Avoid unnecessary JavaScript

## AI Feature (Phase 2 – Optional)
Prepare architecture for future AI features:
- Font recommendation based on platform (Instagram, WhatsApp)
- Text style suggestions
- Save favorite fonts for logged-in users

AI features should be optional and scalable (SaaS-ready).

## Monetization (Future Ready)
- Free users: basic fonts
- Paid users: premium fonts, saved history, no ads
Do NOT implement payment now, just keep structure scalable.

## Legal Pages (Must Be Separate Pages)
- /impressum
- /datenschutz
- /agb

German legal compliance is mandatory.

## Tech Preferences
- Clean, readable code
- SEO-friendly rendering
- Mobile-first UI
- Framework choice left flexible (Next.js / React preferred)

## Things to Avoid
- No English text in UI
- No keyword stuffing
- No desktop-only design
- No missing Impressum
- No slow animations
