# Font Files Directory

This directory contains locally hosted font files to avoid network dependencies during build time.

## Fonts Used:
- **Inter**: Modern sans-serif font (variable font)
- **Pacifico**: Cursive/script font
- **Great Vibes**: Elegant script font
- **Dancing Script**: Handwritten-style font

## How to Add Font Files:

1. Download the fonts from Google Fonts:
   - [Inter](https://fonts.google.com/specimen/Inter)
   - [Pacifico](https://fonts.google.com/specimen/Pacifico)
   - [Great Vibes](https://fonts.google.com/specimen/Great+Vibes)
   - [Dancing Script](https://fonts.google.com/specimen/Dancing+Script)

2. Download the `.woff2` format for best performance

3. Place them in this directory with these names:
   - `Inter-Variable.woff2`
   - `Pacifico-Regular.woff2`
   - `GreatVibes-Regular.woff2`
   - `DancingScript-Variable.woff2`

**Note:** Font files are not committed to git to keep the repository size small.
